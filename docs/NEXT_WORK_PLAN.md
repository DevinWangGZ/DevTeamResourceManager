## 三、待实现功能

### 🔴 第一优先级：智能候选人推荐（P2剩余）

**目标**：基于任务需求技能，自动推荐最适合的开发人员

**工作量**：3-4周

**任务分解**：

**后端任务**（1-2周）：
- [ ] 设计技能匹配算法（技能名称匹配 + 熟练度加权）
- [ ] 实现候选人评分服务
  - [ ] 技能匹配度评分
  - [ ] 当前负荷权重（负荷低的开发者优先）
  - [ ] 历史完成率权重（可选）
- [ ] 实现推荐API端点 `GET /api/v1/tasks/{task_id}/recommend-assignees`

**前端任务**（1周）：
- [ ] 在任务派发对话框中集成推荐列表
- [ ] 展示候选人信息（姓名、匹配技能、当前负荷状态）
- [ ] 支持一键派发给推荐人选

**验收标准**：
- [ ] 推荐结果与任务所需技能相关
- [ ] 考虑开发人员当前负荷
- [ ] 推荐列表按匹配度排序
- [ ] 支持忽略推荐，手动选择
